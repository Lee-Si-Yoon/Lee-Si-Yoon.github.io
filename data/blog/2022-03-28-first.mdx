# Github + Jekyll 블로그 만들기

### [테디노트님 영상](https://www.youtube.com/watch?v=ACzFIAOsfpM)을 보고 따라 만들었습니다.

1. 기본 테마 fork 해오기  
   [테마모음](https://github.com/topics/jekyll-theme)에서 마음에 드는 테마를 선택하고 깃허브로 Fork

2. repository name 바꾸기  
   Fork된 게시물에서 리포지토리 이름을 자신의 "닉네임 + github.io" 으로 수정

3. config.yml 수정하기  
   config.yml에서 site settings의 url부분을 "https://(닉네임).github.io" 으로 수정

4. 첫번째 게시물 올리기
   리포지토리에 `_post`라는 폴더를 만들고 `yyyy-mm-dd-(your name)`형식으로 제목을 붙인다. 그리고
   [Jekyll](https://jekyllrb.com/docs/posts/)을 참조하여

```
---
layout: post
title:  "Welcome to Jekyll!"
---
```

을 서두에 붙이고 글을 쓴다.

5. 마크다운 공부하기
   [HEROPY](https://heropy.blog/2017/09/30/markdown/)
   [TEDDYBEAR](https://teddylee777.github.io/jekyll/Jekyll-%EC%82%AC%EC%9A%A9%EC%9D%84-%EC%9C%84%ED%95%9C-markdown-%EB%AC%B8%EB%B2%95)

6. 마크다운 에디터
   [Typora](https://typora.io/)
   [Marktext](https://github.com/marktext/marktext)
